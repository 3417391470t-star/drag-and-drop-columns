
# Drag and Drop Columns

**显式的行布局工具，支持通过拖拽、缩放来灵活调整内容，并支持每行最多 3 个区块。**



## 功能

-🧱 显式行布局 — 提供独立的行布局容器，严格限制每行最多支持 3 个区块。

-📐 拖拽重排与建行 — 拖拽卡片顶部的 ☰ 手柄可移动该卡片。 将卡片拖放至行与行之间，可直接创建新行。

-📏 卡片自由缩放 — 提供专用的缩放手柄，支持自由拖动来调整卡片的宽度和高度。

-✏️ 即时编辑与预览 — 支持为每张卡片独立设置图标、标题和副标题。 点击卡片正文区域即可切换为文本输入框，原生支持 Markdown、数学公式以及中文输入。

-🎨 丰富的色彩与外观定制 — 内置 9 种精美的色彩面板预设（包含背景色、标题色、文本色、强调色等）。

-🗑️ 安全删除 — 采用双击 × 按钮的机制来删除特定卡片，可有效防止误删。

## 使用

1.插入布局块 — 使用命令面板运行 Insert editable drag layout block 命令即可快速插入布局。 此外，您也可以手动创建 drag-layout 或 drag-cols 类型的代码块来渲染布局。

2.移动与排列 — 拖动卡片内部的 ☰ 手柄（手柄会提示当前的行号与区块编号）即可在行内或跨行移动区块。

3.编辑内容 — 在卡片的上方输入框直接填写 Icon（图标）、Title（标题）和 Subtitle（副标题）。 点击 Body（正文）区域进入编辑模式。

4.调整大小 — 使用鼠标拖拽卡片上的缩放手柄，可调整卡片尺寸。

5.添加与删除 — 点击每行末尾的 + 按钮可以在当前行新增一张卡片（行满 3 个后该按钮会被禁用）。 若要移除卡片，请双击卡片内的 × 删除按钮。

## 提交前本地检查

建议在推送前按以下顺序执行，确保与 AutoReview 的检查口径一致：

```bash
npm install
npm run lint:review
npm run typecheck
npm run test
```

`lint:review` 使用了 `--max-warnings=0`，可在本地严格拦截会导致审核失败的问题。
